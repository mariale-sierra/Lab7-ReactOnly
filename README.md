# LAB7 React Only
Ale Sierra #24405

## Descripción
La aplicación está estructurada en componentes reutilizables y maneja el estado del juego mediante hooks como useState y useEffect.
El jugador controla una serpiente que se mueve dentro de un tablero. Al comer comida, la serpiente crece y el puntaje aumenta. El juego termina cuando la serpiente colisiona con los bordes del tablero.

## Instrucciones para jugar
- Abre el archivo index.html en tu navegador.
- Usa las teclas de flecha del teclado:
  ⬆️ Arriba
  ⬇️ Abajo
  ⬅️ Izquierda
  ➡️ Derecha
- Controla la serpiente para comer la comida (cuadrado rojo).
- Cada vez que comas, la serpiente crecerá y aumentará tu puntaje.
- Evita chocar contra los bordes del tablero, ya que eso terminará el juego.
